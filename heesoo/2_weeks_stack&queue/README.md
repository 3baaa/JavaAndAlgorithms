# Stack & Queue
## Stack : 한쪽 끝에서만 자료를 넣고 뺄 수 있는 LIFO(Last In First Out) 형식의 자료구조
          -LinkedList로 구현 가능
## Queue : 한쪽 끝에서는 삽입연산만 이루어지며 다른 한쪽 끝에서는 삭제연산만이 이루어지는 FIFO(First In Fist Out) 형식의 자료구조
          -일차원배열(원형 큐), LinkedList로 구현가능
